# Intermediate Python Programs

This folder contains **intermediate-level Python programs** that apply
core concepts to real-world problems.

These programs focus on:
- File handling
- Exception handling
- Games and simulations
- Matrix operations

---

## Programs Included

- Student class (Store marks and name)
- Bank management system
- Inventory management system
- Contact book application
- Tic-Tac-Toe game
- Hangman game
- Quiz game
- Matrix operations
- Tower of Hanoi
- Exception handling



## Learning Outcome
By working with these programs, learners will:
- Understand classes and objects
- Apply Python to real-world problems
- Improve problem-solving and code structure
